# Oregon Business Data Project

This project analyzes public Oregon business entity data to extract and structure useful information for research purposes.

## What I did
- Pulled data from Oregon public API (data.oregon.gov)
- Processed and cleaned raw business records
- Extracted owner names and address information
- Structured data for easier analysis in Google Sheets

## Tools
- JavaScript (Google Apps Script)
- Google Sheets
- Public API (Socrata)

## Purpose
This project was developed while exploring business data relevant to real estate research.
